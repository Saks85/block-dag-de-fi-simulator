# BlockDAG DeFi Simulator

[![Deployed on Vercel](https://img.shields.io/badge/Deployed%20on-Vercel-black?style=for-the-badge&logo=vercel)](https://v0-block-dag-de-fi-simulator.vercel.app/)

## 🚀 Overview

The **BlockDAG DeFi Simulator** is an interactive platform that demonstrates how **BlockDAG technology** can supercharge decentralized finance (DeFi).  
Instead of sequential block confirmations like traditional blockchains, BlockDAG allows **parallel transaction processing**, leading to higher throughput, faster confirmations, and scalability for real-world DeFi use cases.

This project was built for **HackOdisha 5 (2024)** under the BlockDAG problem statement.

---

## 🎯 Features

- **Interactive Transaction Input** – Add custom transactions or use pre-set DeFi actions (Trading, Lending, Staking, Liquidity).  
- **Live DAG Visualization** – See blocks and transactions grow dynamically in a directed acyclic graph.  
- **Parallel Processing Simulation** – Demonstrates how multiple transactions can be confirmed at once.  
- **Auto Mode** – Continuously generates transactions like a live DeFi network.  
- **High Volume Stress Test** – Simulates thousands of transactions to measure scalability.  
- **Block Inspector** – Inspect details of any block (transaction type, confirmation status, timestamps).  
- **Metrics Dashboard** – Live counters for Transactions per Second (TPS) and confirmation speeds.
